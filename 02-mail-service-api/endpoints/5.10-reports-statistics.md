# 5.10 Reports & Statistics

## `GET /v1/reports/mails`

Laporan surat terkirim (hanya root mails / surat baru, bukan reply).

**Role**: MAIL_ADMIN atau SUPER_ADMIN

**Query Parameters:**

| Param | Type | Default | Keterangan |
|---|---|---|---|
| `startDate` | date | Awal bulan ini | |
| `endDate` | date | Hari ini | |
| `keyword` | string | — | |
| `typeId` | int | — | |
| `categoryId` | int | — | |
| `page` | int | 1 | |
| `size` | int | 20 | |
| `sort` | string | `id,asc` | |

**Response `200 OK`:**
```json
{
  "success": true,
  "data": [
    {
      "id": 180001,

      "subject": "Rapat Koordinasi Bulanan",
      "createdBy": {
```

## `GET /v1/reports/mails/export`

Export laporan surat ke Excel (XLS).

**Query Parameters:** Sama dengan `GET /v1/reports/mails` (tanpa pagination).

Tambahan:

| Param | Type | Keterangan |
|---|---|---|
| `columns` | string | Comma-separated column keys. Contoh: `mailNumber,date,type,category,subject,createdBy` |

**Response:** Binary stream dengan header:
```
Content-Type: application/vnd.ms-excel
Content-Disposition: attachment; filename="laporan_komunikasi_2026-03-12.xls"
```

## `GET /v1/reports/statistics/by-category`

Statistik jumlah surat per kategori per bulan.

**Query Parameters:**

| Param | Type | Default |
|---|---|---|
| `period` | string (YYYYMM) | Bulan ini |

**Response `200 OK`:**
```json
{
  "success": true,
  "data": [
    {
      "period": "202603",
      "categoryId": 86,
      "categoryName": "Publikasi",
      "total": 15
    }
  ]
}
```

---
