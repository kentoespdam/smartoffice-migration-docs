# 5.1 Folders

## `GET /v1/folders`

List semua folder milik current user (system folders + custom folders) beserta unread counter.

**Response `200 OK`:**
```json
{
  "success": true,
  "data": [
    {
      "id": 2,
      "parentId": 0,
      "name": "Inbox",
      "type": "SYSTEM",
      "iconClass": "inbox",
      "unreadCount": 12,
      "readCount": 45
    },
    {
      "id": 3,
      "parentId": 0,
      "name": "Draft",
      "type": "SYSTEM",
      "iconClass": "draft",
      "unreadCount": 0,
      "readCount": 3
    },
    {
      "id": 15,
      "parentId": 10,
      "name": "Proyek ABC",
      "type": "PERSONAL",
      "iconClass": "email",
      "unreadCount": 2,
      "readCount": 8
    }
  ]
}
```

---

## `POST /v1/folders`

Buat custom folder baru.

**Request Body:**
```json
{
  "parentId": 10,
  "name": "Proyek ABC"
}
```

**Response `201 Created`:**
```json
{
  "success": true,
  "message": "Folder berhasil dibuat",
  "data": {
    "id": 15,
    "parentId": 10,
    "name": "Proyek ABC",
    "type": "PERSONAL",
    "iconClass": "email"
  }
}
```

**Error `409 Conflict`:**
```json
{
  "success": false,
  "errorCode": "FOLDER_DUPLICATE_NAME",
  "message": "Anda sudah memiliki folder dengan nama [Proyek ABC], silahkan pilih nama yang lain"
}
```

---

## `PUT /v1/folders/{folderId}`

Rename custom folder. Hanya folder personal (owner_id = current user).

**Request Body:**
```json
{
  "name": "Proyek ABC - Selesai"
}
```

**Response `200 OK`:**
```json
{
  "success": true,
  "message": "Folder berhasil diperbaharui"
}
```

---

## `DELETE /v1/folders/{folderId}`

Soft-delete custom folder. System folders tidak bisa dihapus.

**Response `204 No Content`**

**Error `422`:** Jika mencoba hapus system folder.
