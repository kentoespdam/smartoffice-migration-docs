# 5.5 Mail Archive (Arsip)

## `GET /v1/archives`

List arsip surat (paginated).

**Query Parameters:**

| Param | Type | Required | Default | Keterangan |
|---|---|---|---|---|
| `startDate` | date | — | 30 hari lalu | Filter tanggal arsip |
| `endDate` | date | — | Hari ini | |
| `keyword` | string | — | — | Search: nomor, subject, tujuan, ref_no, content, keyword |
| `typeId` | int | — | — | Filter mail_type |
| `categoryId` | int | — | — | Filter category |
| `buildingId` | int | — | — | Filter lokasi gedung |
| `floorId` | int | — | — | Filter lokasi lantai |
| `roomId` | int | — | — | Filter lokasi ruangan |
| `page` | int | — | 1 | |
| `size` | int | — | 20 | |
| `sort` | string | — | `mailDate,desc` | |

## `GET /v1/archives/search`

Search arsip untuk user biasa — **hanya arsip yang user punya akses** (berdasarkan position/jabatan). Berbeda dari `GET /v1/archives` yang untuk admin.

**Query Parameters:** Sama dengan `GET /v1/archives` ditambah:

| Param | Type | Keterangan |
|---|---|---|
| `orgId` | int | Filter organisasi |
| `secretType` | string | Filter kerahasiaan |
| `subjectKeyword` | string | Search khusus di subject |
| `contentKeyword` | string | Search khusus di content |
| `archiveKeyword` | string | Search khusus di keyword field |

**Response `200 OK`:** Sama format, ditambah access info:
```json
{
  "success": true,
  "data": [
    {
      "id": 3001,
      "...": "...",
      "myAccess": {
        "canView": true,
        "canDownload": true,
        "canViewHistory": false
      }
    }
  ]
}
```

## `POST /v1/archives`

Buat arsip baru.

**Request Body:**
```json
{
  "mailDate": "2026-03-01",
  "typeId": 1,
  "categoryId": 86,
  "categoryCode": "485.1",
  "orgId": 10,
  "orgCode": "HRD",
  "refId": 180001,
  "refNumber": "001/I/HRD-485.1/III/2026",
  "sentTo": "Kementerian BUMN",
  "subject": "Laporan Tahunan 2025",
  "content": "<p>Detail isi surat...</p>",
  "note": "Arsip asli",
  "status": "DRAFT",
  "secretType": "Biasa",
  "location": {
    "buildingId": 1,
    "floorId": 5,
    "roomId": 12,
    "rack": "R-01",
    "tier": "T-03",
    "box": "B-05"
  },
  "keywords": "laporan, tahunan, 2025"
}
```

**Response `201 Created`:**
```json
{
  "success": true,
  "data": {
    "id": 3002,
    "archiveNumber": "002/I/HRD/III/2026"
  }
}
```

---

## `PUT /v1/archives/{archiveId}`

Update arsip.

**Request Body:** Sama format dengan POST (tanpa auto-generated fields).

**Response `200 OK`:**
```json
{
  "success": true,
  "data": {
    "id": 3002,
    "archiveNumber": "002/I/HRD/III/2026"
  }
}
```

---

## `POST /v1/archives/{archiveId}/finalize`

Finalize arsip (status DRAFT → ARCHIVED). Triggers notification ke user yang punya akses.

**Response `200 OK`:**
```json
{
  "success": true,
  "message": "Arsip berhasil difinalisasi"
}
```

---

## `DELETE /v1/archives/{archiveId}`

Soft-delete arsip (status → DELETED). Jika ada reference ke mail, release `m_ma_id`.

**Response `204 No Content`**
