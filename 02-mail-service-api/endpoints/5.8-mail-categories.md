# 5.8 Lookup: Mail Categories

## `GET /v1/mail-categories`

List kategori surat aktif.

**Query Parameters:**

| Param | Type | Required | Keterangan |
|---|---|---|---|
| `typeId` | int | — | Filter by mail type. `0` atau kosong = semua |
| `page` | int | — | |
| `size` | int | — | |

**Response `200 OK`:**
```json
{
  "success": true,
  "data": [
    {
      "id": 86,
      "typeId": 1,
      "code": "485.1",
      "name": "Publikasi",
      "codeAndName": "485.1 - Publikasi",
      "status": "ACTIVE"
    }
  ]
}
```

---

## `POST /v1/mail-categories`

**Role**: MAIL_ADMIN

**Request Body:**
```json
{
  "typeId": 1,
  "code": "485.2",
  "name": "Surat Edaran"
}
```

**Error `409`:** Duplikasi code + typeId.

---

## `PUT /v1/mail-categories/{categoryId}`

**Role**: MAIL_ADMIN

**Request Body:** Sama format.

---

## `DELETE /v1/mail-categories/{categoryId}`

**Role**: MAIL_ADMIN. Soft-delete (status → DELETED).

**Response `204 No Content`**
