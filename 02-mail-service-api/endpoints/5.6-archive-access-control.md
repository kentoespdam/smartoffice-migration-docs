# 5.6 Archive Access Control

## `GET /v1/archives/{archiveId}/access`

List semua access control entries untuk arsip.

**Response `200 OK`:**
```json
{
  "success": true,
  "data": [
    {
      "id": 500,
      "posId": 50,
      "posName": "Kasi SDM",
      "canAccess": true,
      "canDownload": true,
      "canViewHistory": false
    }
  ]
}
```

---

## `PUT /v1/archives/{archiveId}/access`

Set access control (replace all). Kirim full list — server akan delete existing lalu insert semua.

**Request Body:**
```json
{
  "entries": [
    { "posId": 50, "canAccess": true, "canDownload": true, "canViewHistory": false },
    { "posId": 51, "canAccess": true, "canDownload": false, "canViewHistory": false }
  ]
}
```

**Response `200 OK`:**
```json
{
  "success": true,
  "message": "Access control updated"
}
```

**Side Effect (Async):** Insert `mail_archive_notif` → background job kirim internal mail notification ke semua user di jabatan yang diberi akses baru.
