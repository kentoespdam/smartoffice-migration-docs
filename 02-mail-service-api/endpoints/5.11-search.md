# 5.11 Search

## `GET /v1/mails/search`

Search surat di **semua folder** milik current user (kecuali permanent deleted).

**Query Parameters:**

| Param | Type | Required | Keterangan |
|---|---|---|---|
| `keyword` | string | ✅ | Search: subject, content, note, sender, recipient, no surat, category, type |
| `page` | int | — | |
| `size` | int | — | |
| `sort` | string | — | Default: `createdDate,desc` |

**Response `200 OK`:**
```json
{
  "success": true,
  "data": [
    {
      "id": 180001,
      "mailNumber": "001/I/HRD-485.1/III/2026",
      "date": "2026-03-10",
      "subject": "Rapat Koordinasi Bulanan",
      "createdBy": { "name": "Budi Santoso", "empId": 456 },
      "readStatus": "READ",
      "folderId": 4,
      "folderName": "Read Items",
      "type": { "id": 1, "name": "Internal" },
      "category": { "code": "485.1", "name": "Publikasi" }
    }
  ],
  "meta": {
    "page": 1,
    "size": 20,
    "totalItems": 5,
    "totalPages": 1
  }
}
```

## `GET /v1/mails/find`

Cari surat terkirim (root only) untuk keperluan referencing (misal: arsip, RAB). Beda dengan search — ini search semua surat **global** (bukan per user).

**Query Parameters:**

| Param | Type | Keterangan |
|---|---|---|
| `keyword` | string | Search: no, subject, content, note, creator |
| `typeId` | int | Filter type |
| `categoryId` | int | Filter category |
| `availableFor` | string | `"archive"` → hanya surat yang belum diarsipkan. `"rab"` → belum di-link ke RAB |

**Response `200 OK`:**
```json
{
  "success": true,
  "data": [
    {
      "id": 180001,
      "mailNumber": "001/I/HRD-485.1/III/2026",
      "date": "2026-03-10",
      "createdBy": "Budi Santoso",
      "subject": "Rapat Koordinasi Bulanan"
    }
  ]
}
```
