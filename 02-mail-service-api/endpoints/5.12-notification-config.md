# 5.12 Notification Config (Admin)

## `GET /v1/admin/notification-config`

**Role**: SUPER_ADMIN

**Response `200 OK`:**
```json
{
  "success": true,
  "data": {
    "emailNotifEnabled": true,
    "emailNotifMethod": "IMMEDIATE",
    "senderEmail": "eoffice.notification@gmail.com",
    "senderAlias": "eOffice Notification"
  }
}
```

---

## `PUT /v1/admin/notification-config`

**Role**: SUPER_ADMIN

**Request Body:**
```json
{
  "emailNotifEnabled": true,
  "emailNotifMethod": "QUEUE",
  "senderEmail": "noreply@company.com",
  "senderAlias": "SmartOffice"
}
```

`emailNotifMethod`: `"IMMEDIATE"` (kirim langsung via SMTP) atau `"QUEUE"` (simpan di queue saja, proses background).
