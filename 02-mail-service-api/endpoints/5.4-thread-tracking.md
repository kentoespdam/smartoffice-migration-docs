# 5.4 Thread / Tracking

## `GET /v1/mails/{mailId}/thread`

Ambil seluruh chain surat dalam satu thread (dari root sampai semua replies). Dimulai dari `m_root_id` surat ini.

**Query Parameters:**

| Param | Type | Default | Keterangan |
|---|---|---|---|
| `sort` | string | `createdDate,asc` | |
